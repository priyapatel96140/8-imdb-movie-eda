# IMDB Movie Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the IMDB Movie Dataset to uncover insights related to movie performance, audience engagement, ratings, revenue, and production trends.

The analysis focuses on understanding how factors such as runtime, ratings, votes, directors, genres, and release years influence the success of movies.


## Business Problem

Movie production companies and distributors need to understand the factors that contribute to a movie's success.

This project aims to answer questions such as:

* Do longer movies receive better ratings?
* Which years generated the highest revenue?
* Which years received the highest audience engagement?
* Which directors consistently produce highly rated movies?
* Which genres are most common?
* Is there a relationship between ratings and revenue?


## Dataset Information

The dataset contains information about 1000 movies including:

* Rank
* Title
* Genre
* Description
* Director
* Actors
* Year
* Runtime (Minutes)
* Rating
* Votes
* Revenue (Millions)
* Metascore
